# NVIDIA Stock Return Analysis

## Overview

This project analyzes 10 years of NVIDIA stock performance using financial data, market benchmarks, and investor attention metrics.

## Tools Used

* Python
* Pandas
* NumPy
* Statsmodels
* Bloomberg Terminal
* Tableau
* Google Trends API

## Key Analysis

* Correlation Analysis
* Regression Modeling
* Market Benchmark Comparison
* Revenue Growth Analysis
* Investor Attention Analysis

## Key Findings

* Technology sector performance was the strongest predictor of NVIDIA stock returns
* Regression model explained approximately 56% of quarterly return variation
* Google Trends showed delayed investor reaction behavior
