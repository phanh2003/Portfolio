# Sales Performance Dashboard

## Overview

This project uses Tableau dashboards to analyze business sales performance, customer behavior, and regional revenue trends.

## Tools Used

* Tableau
* Excel
* Business Analytics

## Dashboard Features

* Quarterly sales analysis
* Customer segmentation
* Revenue trend visualization
* KPI reporting
* Regional performance tracking

## Key Insights

* Identified top-performing product lines and regions
* Evaluated quarterly revenue performance trends
* Analyzed customer purchasing behavior and category performance
