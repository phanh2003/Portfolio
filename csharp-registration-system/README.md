# C# Registration System

## Overview

This project is a C# Windows Forms application designed to simulate a registration management system with interactive user interface components and form-based workflows.

## Tools Used

* C#
* Windows Forms
* Visual Studio
* .NET Framework

## Application Features

* User registration form
* Form validation
* Multi-form navigation
* Interactive graphical user interface
* Structured event-driven programming

## Skills Demonstrated

* Object-Oriented Programming (OOP)
* GUI Development

* Form Design
* Application Logic Development
* C# Programming
