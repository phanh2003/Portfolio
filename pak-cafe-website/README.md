# PAK Cafe & Bakery Website

## Overview

This project is a multi-page business website created for PAK Cafe & Bakery, featuring Vietnamese-inspired drinks and bakery products.

## Tools Used

* HTML
* Basic Web Design
* Business Branding

## Website Features

* Multi-page website navigation
* Product descriptions and pricing
* Menu presentation
* Business branding and layout design
* Customer-focused content structure

## Featured Products

* Coconut Coffee
* Salty Coffee
* Vietnamese Green Rice Coconut Milk Slush
* Vietnamese Pandan Hollow Donut
* Palm Sugar Honeycomb Cake

## Skills Demonstrated

* HTML Development
* Website Structure Design
* Product Marketing Presentation
* Business-Oriented Web Content
* Front-End Fundamentals

## Project Files

* HTML pages
* Product images
* Menu design elements
